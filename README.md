椰风纸语——海口秀英剪纸数字化传承平台
一个用海口秀英剪纸文化带动火山荔枝销售的PC端网页。

背景
秀英剪纸是海口本地的非遗项目，但知道的人不多。火山荔枝是当地的特色农产品，品质不错，但缺一些有吸引力的包装和故事。这个项目想做一个尝试：把剪纸的视觉元素和荔枝结合起来，用文化内容带动销售。

主要功能
首页：展示平台介绍、快捷入口和推荐内容，已更新Logo为新的IP形象
课程页：浏览和筛选在线课程
商城页：选购文创产品和火山荔枝
个人中心：管理我的课程、订单和收藏
助农专区：了解农户故事和购买助农产品，已更新为您提供的本地图片
管理端：完整的后台管理系统，包含仪表盘、课程管理、商品管理、订单管理、用户管理、助农管理和系统设置
